About ERICA

Hydra uses ERICA Tool's data and approaches for calculations.

The ERICA Tool is a software system that allow user to perform calculations of dose rate to non-human biota.

More about ERICA on it's website.

How to use

  1. Drag and drop organisms in media.
  2. Enter specific activities of radionuclides. Hydra will calculate missing values using ERICA's coefficients.
  3. Push Calculate button.
  4. Get results under app frame.
